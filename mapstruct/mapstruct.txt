краткое описание 

- [ mapstruct ] java библиотека для намисания мапперов(mapper).



1) настраиваем
  - добавляем в проект
    [
      compileOnly "org.mapstruct:mapstruct:${mapstruct_version}"
      annotationProcessor "org.mapstruct:mapstruct-processor:${mapstruct_version}"
    ]


2) используем
  - [*] надо договориться об одном стиле написания. (*) можно посмотреть пример!
  - кажется удобносделать так
    - Один класс маппера предназначен для превращения чего угодно в этот класс.
    - Имя класса это 
      <корень проекта>.mapper.<дублируем пакеты класса>.<класс в который делаем памминг>Mapper
      получается в пакете mapper создаются по тем же путям классы мапперов.
    - используем такой порядок полей аннотации, чтобы визуально легко читать маппе
      [
        @Mapping(target = "fileUuid", source = "dto.file.uuid")
        FileTransferHistory map(MessageFromDboDto dto);
      ]
