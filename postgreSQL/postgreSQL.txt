краткое описание 

- [postgresql] это реляционная база данных КЛАССИКА ! выбирай её !




1) устанавливаем 
  - устанавливаем [ sudo apt install posqresql ]
  - устанавливаем [ sudo apt install postgresql-client ]
  - устанавливаем [ sudo apt install pgadmin3 ]
  

2) подключаемся
  - пробуем зайти на локальную базу
    - [ sudo netstat -natp | grep postgres ] получаем [ 5432 ]
  - По умолчанию соединения через TCP/IP заблокированы.
    - топаем [ cd /etc/postgresql/9.5/main/ ]
    - открываем [ sudo vim ./postgresql.conf ]
    - раскоменчиваем [ listen_address ] и можем поставить '0.0.0.0'
    - заходим вот так, заметь что ключ для sudo
      - заходим [ sudo -u postgres psql ]    [*] поменяй на одинарные ковычки
        - есл видим ошибку надо [ctrl+c] чтобы было не [ =# а -# ]
        - меняем пароль [ ALTER USER "postgres" with encrypted password 'postgres' ; ]
        - ставим пароль [ postgres ]
      - в [ sudo vim /etc/postgresql/8.4/main/pg_hba.conf ] ставим использование [ md5 ]
      - [ host     all         postgres     0.0.0.0/0           md5 ]
  - можем перезагрузить сервис [ sudo /etc/init.d/postgresql restart ] 
    - теперь заходим по паролю и можем попасть через [ pgadmin3 ]
      - [ psql -h localhost -p 5432 -U postgres ]


3) Simple commands
  - [ \l ] or [ \list ] - list all available databases
  - [ \c <database_name> ] - Connect to a different database
  - [ \dn ] - Show All Schemas
  - [ \dt ] - list all tables in the currently connected database 
  - [ \d <table_name> ] - to describe the structure of a specific table (columns, types, etc.)
  - [ SELECT * FROM users; ] example of sql
