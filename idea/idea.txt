краткое описание 

- [idea] супер пупер IDE




1) устанавливаем  
  - качаем с офф сайта
  - разархивируем [ tar -xzf idea-2017.2.3.tar.gz ]
  - запускаем [ idea.sh ]
  - [!] не работают удобные HotKeys. Тогда возможно 
    их перехватывает операционная система
    - [*] идем в настройки операционной системы
      [клавиатура] -> [находим те HotKes] и отключаем


2) обновляем [idea]
  - из среды нас кидают на самую свежую версию для скачивания
  - качаем 
  - топаем туда где валяется старая идея [ /home/dzhukov/dounload/idea ]
    - удаляем стар архив и папку
    - разархивируем [ tar -xzf idea-2017.2.3.tar.gz ]
    - запускаем [ idea.sh ]


3) качаем JDK
  - топаем на сайт Oracle 
    - [ http://www.oracle.com/technetwork/java/javase/downloads/index.html ]
  - распаковываем [ tar -xzf ]
  

4) Плагины (Plagins)
  - [ sonarlint ] статический анализатор кода
    - [ SonarQube for IDE ] looks like they rename a plugin
  - [ Snyk ] real-time vulnerability scanning of code, open source libraries, 
    containers, and cloud infrastructure — and provides actionable fix advice in-line 
    so you can fix quickly and move on.
  - [ Codota ] прокаченное автодополнение кода + можно посмотреть примеры кода
  - [ Lombok ] нужен чтобы удобно работать с проектом]. использую Lombok
  - [ Kubernates ] упрощает работу с kubernates файлами
  - [ Go Template ] в helm используется этот шаблонизатор по умолчанию
  - [ OpenAPI Specifications ] Для просмотра Swagger Файлов
  - [ Makefile Language ] можно будет завести  MakeFile и запускать такски 
  - [ AsyncAPI ] как Swagger но для ассинхронного взаимодействия
  - [ PlantUML Integration ] Для работы с *.pumpl файлами
    - gradle plagin for doing fomething cool with *pumpl
    - [ id("com.cosminpolifronie.gradle.plantuml") ]
  - [ AsciiDoc ] Это формат подходит для Confluense (post docs after build )
    -(*) publichser plugin: id("ch.nomisp.confluence.publisher")
  - [ ktlint ] Checks or do code auto format
    - link: https://github.com/JLLeitschuh/ktlint-gradle
    - link: https://github.com/pinterest/ktlint
    - comands in gradle:
      - [ gradle ktlint ] run the verification
      - [ gradle ktlintFormat ] run the formatter
  - [ Terraform and HCL ] for terraform support
  - [ GitHub Actions Manager ] for github actions
  

5) Хоткеи в маке ( Mac OS )
  - [ Shift x2 ], поиск по классам
  - [ command + f ], обычный поиск
  - [ comand + шифт + f ], поиск по всему проекту
  - [ command + r ], замена
  - [ command + option + стрелочка ], перемещаемся по проекту
  - [ command + ЛКМ ], проваливается внутрть класса или метода
  - [ command + n ], генератор конструкторов и гетеров сетеров
  - [ option + enter ], выполнить подсказку идеи
  - [ control + space ], автодополнение (*) отключили изменение языка на это сочетание
