краткое описание 

- MySQL есть в репозиториях Ubuntu. Он разбит на несколько пакетов.




1) устанавливаем и удаляем)))
  - устанавливаем сервер [ sudo apt install mysql-server ]
    - (*) попросят установить пароль для [root]
      (*Вход) [mysql -u root -p] 
  - хотим посмотреть конфигурационный файл:
      [/etc/mysql/mysql.conf.d/mysqld.cnf]
      - меняем слушать конекты от всех IP 
        [bind-address = 0.0.0.0]
      - перезагружаем демона [/etc/init.d/mysql restart]
      //- еще надо отключить [fierwall] -> [ufw disable] -> [reboot]
      - создаем пользователя
        [ CREATE USER 'dzhukov'@'%' IDENTIFIED BY 'dzhukov' ]
      - даем все привелегии
        [GRANT ALL PRIVILEGES ON * . * TO 'dzhukov'@'%']
      - теперь к нам можно подключиться с другого сервера 
        под пользователем [dzhukov] 
  - устанавливаем клиент [sudo apt install mysql-client]
  - после настройки сервера можем попробовать подключиться:
    [ mysql -P 3306 -h 192.168.56.11 -u dzhukov -p ]
  - [!] Были различного рода проблемки после некорректного
    завершения работы сервера с работающим [mysql]
    так-как установили с сайта через команду [ apt install ./{имя пакета.deb} ]
    - [*] Хотим удалить все и заново поставить
      - [ sudo apt remove mysql-server* ] # просто их несколько
      - [ sudo rm -r /usr/lib/mysql* ]    # несколько папочек
 

2) хотим установить [mysql-workbench]
  - скачали [mysql-workbench-community-6.3.9-1ubuntu16.04-amd64.deb]
  - установили [ apt install ./{имя пакета.deb} ]
