краткое описание 

- [nodejs] это инторпритатор языка [ javaScript ]. Получаем на вход
  код, написанный на [ javaScript ] и выполняем его.

- [npm] это пакетный менеджер пакетов для [ nodejs ]

- [*] Очень старые версии пакетов в репозитории [ Ubuntu ]
  Поэтому руководствуемся инструкциями [ nodejs.org ]




1) устанавливаем  
  - качаем [ curl -sL https://deb.nodesource.com/setup_8.x | sudo -E bash - ] 
    - качает пакет в кеш пакетного менеджера [ apt ]
  - устанавливаем [ sudo apt-get install -y nodejs ]
  - проверяем версию [ node -v ]
  - проверяем интерпритатор
    - запускаем [ nodejs ]
    - выполняем [ console.log("Hello World"); ]


2) у нас установлен nvm
  - это удобная штука чтобы иметь несколько версий ноды
  - смотрим доступные версии для установки
    - [ nvm  ls-remote ]
  - устанавливаем последнюю LTS версию [ nvm install v10.15.1 ]
  - переключаемся на скаченную версию [ nvm use v10.15.1 ]
  - проверяемс [ node -v ]
