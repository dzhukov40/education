- [skype] позволит нам общаться




1) Skype — программа с закрытым исходным кодом, а поэтому она просто 
   не может находиться в официальных репозиториях Ubuntu. Но она, 
   как и другие проприетарные программы есть в репозитории партнеров 
   Cnanonical. Это репозиторий не включен по умолчанию, поэтому его 
   нужно подключить.
 
2) Устанавливаем (*) не нашелся пакет ((
   - Добавим репозиторий партнеров    
     [ sudo add-apt-repository "deb http://archive.canonical.com/ $(lsb_release -sc) partner" ]
   - Добавим архитектуру i386
     [ sudo dpkg --add-architecture i386 ]
   - Обновляем и удаляем
     [ sudo apt update && sudo apt install skype ]


3) Ручная установка (*) вышло, потом запускаем из терминала [ skypeforlinux ]
   - качаем deb пакетик
     [ https://www.skype.com/ru/download-skype/skype-for-linux/ ]
   - устанавливаем, запуская в папке где пакет
     [ sudo dpkg -i skype-ubuntu-precise_4.3.0.37-1_i386.deb ]
